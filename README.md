# DATA-ANALITYS-CREACION-DE-DIAGRAMA-DE-PRODUCCION
El diagrama entidad‑relación de Producción organiza las tablas clave vinculadas al producto: inventario, órdenes de trabajo, costos, documentos y transacciones. Muestra cómo el producto es la entidad central y permite estructurar datos confiables para análisis en SQL Server y Power BI.
